=========
rio-cogeo
=========

CloudOptimized GeoTIFF creation plugin for rasterio


Install
=======

.. code-block:: console

    $ pip install -U pip
    $ pip install rio-cogeo

Or install from source:

.. code-block:: console

    $ git clone https://github.com/mapbox/rio-cogeo.git
    $ cd rio-cogeo
    $ pip install -U pip
    $ pip install -e .
